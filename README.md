# Security Policy Writing Lab

## What I’m learning
How organizations create security policies to protect data and systems.

---

## Password Policy Example

- Minimum 12 characters
- Must include numbers and symbols
- Must be changed every 90 days
- MFA required for all accounts

---

## Data Handling Policy

- Do not store PII on personal devices
- Encrypt all sensitive data
- Share data only through approved systems

---

## Why this matters
Policies enforce consistent security behavior across the organization.
